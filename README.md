# ImageGallery_CodeAlpha

A nature image gallery built using HTML, CSS and JavaScript.

## Features
- Next/Previous navigation buttons
- Thumbnail preview images
- Smooth hover effects
- Beautiful dark gradient background

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Click → to go to next image
2. Click ← to go to previous image
3. Click any thumbnail to jump to that image
